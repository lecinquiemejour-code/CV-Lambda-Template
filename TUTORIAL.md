# 🚀 Tutoriel : Créez et lancez votre CV Interactif I.A. (Projet CV IA lambda)

Bienvenue dans l'aventure **CV IA lambda** ! 🌟 

Ce guide est conçu spécifiquement pour vous, même si vous n'avez jamais écrit une ligne de code de votre vie. À la fin de ce tutoriel, vous aurez votre propre site web de CV, propulsé par l'intelligence artificielle la plus puissante de Google, et accessible par tout le monde sur Internet.

---

## 🗺️ Le Plan de votre réussite

Voici les 7 étapes que nous allons franchir ensemble :

1.  **🦾 Étape 1 : Obtenir votre Assistant** (Installation d'Antigravity)
2.  **🛠️ Étape 2 : Préparer ses outils** (Création des comptes gratuits)
3.  **🔱 Étape 3 : Récupérer le projet** (Faire votre propre "photocopie")
4.  **🚀 Étape 4 : Ouvrir dans Antigravity** (Le moment où l'assistant prend les commandes)
5.  **🧭 Étape 5 : Configurer les RULES** (Donner son code de conduite à l'IA)
6.  **🎨 Étape 6 : Personnaliser votre CV** (Remplir vos infos avec l'IA)
7.  **🌍 Étape 7 : Mise au monde** (Déploiement final sur Netlify)

## 🧐 C'est quoi "CV IA lambda" et "Antigravity" ?

---

## 🦾 Étape 1 : Obtenir votre Assistant (Antigravity)

Avant de préparer vos comptes, vous avez besoin de votre copilote. C'est lui qui vous guidera dans toutes les étapes suivantes.

### 1. Activer l'offre Gemini Pro (1 mois offert)
Avant d'installer l'assistant, assurez-vous que votre "cerveau" IA est prêt. Google propose un essai gratuit de 1 mois pour son meilleur modèle.
- **Action** : Allez sur [Gemini](https://gemini.google.com/advanced) et activez l'offre d'essai "Gemini Advanced". Cela liera la puissance de l'IA à votre compte Google.

### 2. Téléchargement et Installation d'Antigravity
Maintenant que votre compte est "surboosté", installez l'outil qui va l'utiliser pour vous.
- **Action** : Téléchargez Antigravity sur [antigravity.google](https://antigravity.google). 
- **Action** : Installez-le et connectez-vous avec le compte Google que vous venez de préparer.

---

## 🛠️ Étape 2 : Préparer ses outils (vos comptes)

Maintenant que votre assistant est prêt à vos côtés, nous allons préparer les trois "clés" pour ouvrir les portes du web. Ces comptes sont tous **gratuits**.

### 1. GitHub : Votre coffre-fort de fichiers
GitHub est l'endroit où votre code va habiter. C'est ici que vous ferez votre "photocopie" (le Fork) du projet.
- **Action** : Rendez-vous sur [github.com](https://github.com/) et inscrivez-vous.

### 2. Netlify : Votre antenne de diffusion
Netlify va prendre vos fichiers sur GitHub et les transformer en un site web vivant accessible par une adresse (URL).
- **Action** : Allez sur [netlify.com](https://www.netlify.com/). Choisissez de vous connecter **avec votre compte GitHub** (c'est plus simple).

### 3. Google AI Studio : Le cerveau de votre site
C'est ici que vous récupérerez la "clé API" (le code secret) qui permettra au chatbot de votre CV de répondre aux recruteurs.
- **Action** : Connectez-vous sur [Google AI Studio](https://aistudio.google.com/) avec votre compte Google habituel.
- **Action** : Cliquez sur **Get API key** puis **Create API key**. Copiez précieusement ce code secret.
- **Précision importante** : Même si vous profitez de l'offre Gemini Pro sur Antigravity, la clé API pour votre futur site fonctionnera sur le **plan gratuit** de Google AI Studio. C'est largement suffisant pour un CV !

> [!IMPORTANT]
> **Où mettre cette clé ?**
> Une fois votre projet ouvert dans Antigravity, dites-moi simplement : *"Voici ma clé API : [Coller votre clé]. Peux-tu l'installer pour moi ?"*. Je m'occuperai de la configurer en toute sécurité.

---

> [!TIP]
> **Pourquoi deux choses différentes ?**
> - **Antigravity Pro** : C'est la puissance pour vous aider à *construire* le CV.
> - **Clé AI Studio Gratuite** : C'est ce qui permet au CV de *parler* tout seul une fois qu'il est en ligne.

## 🔱 Étape 3 : Récupérer le projet (Le Fork)

C'est ici que vous faites votre photocopie personnelle du projet **CV IA lambda**.

1. **Aller sur le modèle** : Ouvrez la page GitHub du projet [github.com/lecinquiemejour-code/CV-Template-Lambda](https://github.com/lecinquiemejour-code/CV-Template-Lambda).
2. **Cliquer sur "Fork"** : En haut à droite de la page, cliquez sur le bouton **Fork**. Cela va créer une copie exacte du projet sous votre propre nom.
3. **Préparer son "Rangement" sur l'ordinateur** : 
   - Avant de télécharger, choisissez (ou créez) un endroit propre sur votre ordinateur. 
   - Par exemple, ouvrez votre explorateur de fichiers et créez un dossier nommé `MonCVinteractif` dans vos Documents. C'est là que nous rangerons votre CV.

4. **Cloner (Télécharger) sur votre ordinateur** : 
   - Sur la page de votre projet (GitHub), cliquez sur le bouton vert **<> Code**.
   - Cliquez sur l'icône "copier" (deux petits carrés) à côté de l'adresse qui finit par `.git`.
   - **Ouvrez Antigravity**.
   - Sur l'écran d'accueil, cliquez sur **Clone Repository** (Cloner un dépôt).
   - Collez l'adresse que vous venez de copier dans la barre qui apparaît.
   - **Important** : Antigravity vous demande "Où enregistrer ?". Sélectionnez le dossier `MonCVinteractif` que vous venez de créer.
   - Cliquez sur **Select as Repository Destination**. L'assistant télécharge tout seul votre CV à l'intérieur !

## 🚀 Étape 4 : Ouvrir dans Antigravity (Le moment crucial)

Une fois le dossier téléchargé, il faut l'ouvrir pour que l'assistant puisse commencer son travail.

- **Action** : Si Antigravity ne l'a pas fait automatiquement, cliquez sur **File** (Fichier) puis **Open Folder** (Ouvrir le dossier).
- **Action** : Sélectionnez le dossier que vous venez de cloner (celui qui s'appelle `CV-IA-lambda`).
- **Résultat** : Antigravity va analyser tout le projet. À partir de cet instant, **je peux enfin vous entendre et agir sur vos fichiers !** Regardez en bas à droite ou à gauche, un petit panneau de discussion (Chat) est prêt à vous accueillir.

## 🧭 Étape 5 : Donner ses instructions à l'IA (Les RULES)

Antigravity est très puissant, mais il a besoin de savoir **comment** vous parler. Pour un débutant, nous allons lui donner des "règles de conduite" (les RULES).

### 1. Pourquoi des RULES ?
Par défaut, une IA peut être trop technique. Les RULES permettent de lui dire : "Je suis un débutant, explique-moi tout simplement et ne fais rien sans mon accord".

### 2. Comment les configurer ?
- **Action** : Dans Antigravity, regardez tout en haut à droite. Cliquez sur le bouton avec les **trois petits points `...`** (Additional Options).
- **Action** : Dans le menu qui s'ouvre, cliquez sur **Customizations** (ou ouvrez l'Agent Manager via l'icône de robot).
- **Action** : Allez dans l'onglet **Rules**.
- **Action** : Cliquez sur **+ Workspace** ou modifiez la partie **Global**.
- **Action** : Copiez et collez le texte suivant (le "Code de conduite" de votre projet) :

```markdown
Ces règles encadrent ton comportement dans ce projet. Elles sont non négociables.

## GARDE-FOUS
### Règle 1 — Checkpoint obligatoire
Ne jamais écrire ou modifier du code sans approbation explicite (“GO”).
### Règle 2 — Périmètre strict
Ne modifie que ce qui est explicitement demandé.
### Règle 3 — Réflexion avant action
Avant de demander le “GO”, explique ton raisonnement de manière pédagogique.

## MÉTHODE DE TRAVAIL
### Règle 4 — Décomposition en sous-tâches
Décompose chaque tâche complexe en étapes petites et séquentielles.
### Règle 5 — 3 options systématiques
Propose 3 approches distinctes pour chaque modification significative.
### Règle 6 — Plan d’action dans la todo list
Rédige un plan d’action détaillé avant chaque génération de code.
### Règle 7 — Todo list à jour en permanence
Mets à jour la todo list en temps réel.

## QUALITÉ DU CODE
### Règle 8 — Simplicité d’abord (KISS)
Privilégie toujours la solution la plus simple.
### Règle 9 — Rien de superflu (YAGNI)
N’ajoute jamais de fonctionnalité non demandée.
### Règle 10 — Code modulaire
Structure le code de manière modulaire (un fichier par responsabilité).
### Règle 11 — Logs de débogage détaillés
Ajoute des console.log explicites à chaque étape clé.
### Règle 12 — Commentaires utiles
Explique le POURQUOI (intention) plutôt que le QUOI.

## POSTURE
### Règle 13 — Communication pédagogique
Explique chaque décision technique en termes accessibles.
```

- **Action** : Enregistrez. Désormais, votre assistant respectera scrupuleusement ces consignes de sécurité.

---

> [!TIP]
> **Le saviez-vous ?** Ces règles sont le "contrat de confiance" entre vous et l'IA. Elles vous garantissent que vous restez le seul maître à bord.

## 🎨 Étape 6 : Personnalisation (Remplir votre CV)

C'est ici que l'aventure devient concrète. Vous n'avez pas besoin d'ouvrir les fichiers compliqués : **je suis là pour le faire à votre place.**

### 6.1 Comment me parler ?
Utilisez la fenêtre de discussion (Chat) en bas de l'écran Antigravity. Parlez-moi comme à un collègue.

> [!TIP]
> **L'astuce "Gain de temps"** 🚀
> Si vous avez déjà un CV classique au format PDF, donnez-le moi tout de suite !
> - **Action** : Faites glisser votre PDF dans le chat et dites : *"Voici mon CV actuel, peux-tu extraire mes informations pour remplir mon nouveau CV interactif ?"*
> - **Résultat** : Je vais lire votre document, extraire vos expériences, vos diplômes et vos compétences pour tout pré-remplir à votre place. Il ne vous restera qu'à vérifier !

### 6.2 Personnaliser votre Identité (`identity.json`)
Toutes vos informations de base (Nom, Prénom, Métier, Liens) sont dans le fichier `identity.json`.
- **Action** : Demandez-moi de mettre à jour votre nom, votre titre professionnel ou vos réseaux sociaux.

### 6.3 Rédiger vos expériences et vos projets
Vos textes longs sont rangés dans des fichiers dédiés :
- **Expériences Professionnelles** : Fichier `experiences.md`.
- **Projets / Portfolio** : Fichier `portfolio.md`.
- **Biographie / Accueil** : Fichier `greeting.md`.

- **Action** : Tapez dans le chat :
  > "Peux-tu mettre à jour mon **portfolio.md** avec ce projet : [Détails du projet] ?"
  > "Modifie mon **experiences.md** pour ajouter mon dernier poste chez [Entreprise]."

> [!TIP]
> **Pas de panique !** Vous n'avez pas besoin d'ouvrir ces fichiers vous-même. En citant leurs noms (comme `portfolio.md`), vous aidez simplement l'IA à être encore plus rapide !

### 6.4 Votre Photo (et votre CV PDF)
Un bon CV a besoin d'une photo de profil.
- **Action** : Préparez votre photo (nommée `photo.jpg`) sur votre bureau.
- **Action** : Dites-moi dans le chat :
  > "Voici ma photo. Peux-tu l'ajouter à mon CV ?"
- **Note pour le PDF** : Si vous me l'avez déjà donné pour l'astuce "Gain de temps" au début de cette étape, **c'est déjà fait !** Je l'ai déjà rangé au bon endroit pour que le bouton "Télécharger mon CV" fonctionne.

### 6.5 Le rituel du "GO"
Pour chaque modification (texte ou fichier), je vais :
1. Vous expliquer ce que je compte faire.
2. Vous montrer les changements.
3. **Attendre votre "GO"** (Règle 1). Tapez simplement "GO" dans le chat pour valider.

### 6.6 Tester votre CV (Le "Check" local)
Avant de le montrer au monde, vérifions que tout fonctionne.
- **Action** : Dans Antigravity, cherchez le bouton **Preview** ou l'icône de **petit globe/navigateur** (souvent en haut à droite).
- **Le Test** : Votre site s'affiche ! Essayez de parler au chatbot. Si vous m'avez donné votre clé API à l'Étape 2, il vous répondra déjà.
- **Pourquoi faire ça ?** Cela vous permet de peaufiner vos textes et de voir le résultat immédiatement, sans attendre la mise en ligne.

## 🌍 Étape 7 : Mise au monde (Le Déploiement)

Votre CV est prêt sur votre ordinateur, mais il doit maintenant être visible par les recruteurs.

### 7.1 Lier GitHub à Netlify et configurer la clé
- **Action** : Allez sur votre compte [Netlify](https://app.netlify.com/).
- **Action** : Cliquez sur **Add new site** > **Import an existing project**.
- **Action** : Choisissez **GitHub**. Sélectionnez votre projet `CV-IA-lambda`.
- **Action** : Avant de cliquer sur Deploy, allez dans **Site configuration** > **Environment variables**.
- **Action** : Cliquez sur **Add a variable** :
  - **Key** : `VITE_GEMINI_API_KEY`
  - **Value** : [Collez ici votre clé API Google AI Studio]
- **Action** : Cliquez sur **Create and Deploy**.
- **Résultat** : Netlify vous donne une adresse (URL). **Votre site est en ligne et son intelligence est activée !**

### 7.2 La boucle vertueuse : Le PDCA
C’est ici que réside la vraie puissance de votre collaboration avec l’IA. Le déploiement n'est pas une fin, mais le début d'un cycle d'amélioration continue appelé **PDCA** :

1.  **P (Plan)** : Vous avez une idée (ex: "Je veux ajouter une section Compétences").
2.  **D (Do)** : Vous me demandez de le faire dans le chat, et vous donnez votre **"GO"**.
3. **C (Check)** : Votre site se met à jour **automatiquement** sur Netlify. 
   - **Comment ça marche ?** Chaque fois que vous me dites "GO" pour une modification, je "pousse" (push) les fichiers vers GitHub. 
   - **Délais** : Dès que je "pousse", Netlify reçoit l'alerte et reconstruit votre site. Cela prend généralement **30 à 60 secondes**.
   - Vous ouvrez le lien et vous vérifiez le résultat.
4. **A (Act)** : Si ce n'est pas parfait, vous me demandez d'ajuster. On recommence !

> [!TIP]
> **Mag...ique ? Non, Logique !** 
> Grâce à cette boucle, votre CV ne sera jamais obsolète. Vous pouvez le faire évoluer en 2 minutes dès que vous avez une nouvelle mission ou un nouveau projet.
